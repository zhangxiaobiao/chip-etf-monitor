# 技术指标参考手册

## 概述
本系统集成了多种技术指标用于分析股票价格走势和生成买卖信号。

## 移动平均线 (Moving Averages)

### SMA (简单移动平均线)
- **SMA 10**: 10日移动平均线，反映短期趋势
- **SMA 20**: 20日移动平均线，反映中期趋势
- **SMA 50**: 50日移动平均线，反映长期趋势

### 信号规则
1. **多头排列**: SMA 10 > SMA 20 > SMA 50 → 买入信号
2. **空头排列**: SMA 10 < SMA 20 < SMA 50 → 卖出信号
3. **价格突破**: 价格突破所有均线 → 强势信号
4. **价格跌破**: 价格跌破所有均线 → 弱势信号

## MACD (移动平均收敛发散)

### 计算公式
- **MACD线**: EMA(12) - EMA(26)
- **信号线**: MACD线的9日EMA
- **柱状图**: MACD线 - 信号线

### 信号规则
1. **金叉**: MACD线从下向上穿越信号线 → 买入信号
2. **死叉**: MACD线从上向下穿越信号线 → 卖出信号
3. **零轴上方**: MACD > 0 → 多头市场
4. **零轴下方**: MACD < 0 → 空头市场

### 参数设置
- 快线周期: 12日
- 慢线周期: 26日
- 信号周期: 9日

## RSI (相对强弱指数)

### 计算公式
RSI = 100 - 100 / (1 + RS)
RS = 平均上涨幅度 / 平均下跌幅度

### 信号规则
1. **超买**: RSI > 70 → 卖出信号
2. **超卖**: RSI < 30 → 买入信号
3. **强势**: RSI > 50 → 多头占优
4. **弱势**: RSI < 50 → 空头占优

### 参数设置
- 计算周期: 14日
- 超买线: 70
- 超卖线: 30

## 布林带 (Bollinger Bands)

### 计算公式
- **中轨**: N日移动平均线
- **上轨**: 中轨 + K × N日标准差
- **下轨**: 中轨 - K × N日标准差

### 信号规则
1. **触及上轨**: 价格触及或突破上轨 → 卖出信号
2. **触及下轨**: 价格触及或跌破下轨 → 买入信号
3. **带宽扩大**: 波动率增加，可能有大行情
4. **带宽收缩**: 波动率减小，可能酝酿突破

### 参数设置
- 周期: 20日
- 标准差倍数: 2.0

## 成交量分析

### 指标
- **成交量**: 当日成交股数
- **成交量均线**: 20日平均成交量
- **量比**: 当日成交量 / 成交量均线

### 信号规则
1. **放量上涨**: 价格上涨伴随成交量放大 → 买入信号
2. **放量下跌**: 价格下跌伴随成交量放大 → 卖出信号
3. **缩量调整**: 价格调整伴随成交量萎缩 → 观望信号
4. **量价背离**: 价格与成交量走势相反 → 警惕信号

## 综合信号生成

### 信号强度计算
系统采用加权评分机制：
- 每个技术指标根据置信度评分 (0-100)
- 买入信号得正分，卖出信号得负分
- 总分 = Σ(各指标分数)
- 信号强度 = 总分 / 总可能分数 × 100%

### 风险等级划分
- **低风险**: 信号强度 ≥ 80，多指标一致
- **中低风险**: 信号强度 60-79，主要指标一致
- **中等风险**: 信号强度 40-59，指标有分歧
- **中高风险**: 信号强度 20-39，信号较弱
- **高风险**: 信号强度 < 20，数据不足

### 投资建议生成
基于信号强度和风险等级生成具体建议：
- 强烈买入/卖出
- 建议买入/卖出
- 考虑买入/卖出
- 轻仓买入/减仓卖出
- 观望

## 参数优化建议

### 不同市场环境
1. **趋势市**: 延长均线周期，增加趋势跟踪权重
2. **震荡市**: 缩短均线周期，增加振荡指标权重
3. **高波动市**: 调整布林带标准差，增加风险控制

### 不同时间框架
1. **日内交易**: 使用分钟级数据，缩短指标周期
2. **短线交易**: 使用日线数据，标准参数设置
3. **长线投资**: 使用周线数据，延长指标周期

## 指标组合策略

### 保守型策略
- 优先考虑：均线排列 + MACD
- 辅助指标：RSI + 布林带
- 要求：至少3个指标一致才生成信号

### 激进型策略
- 优先考虑：突破信号 + 成交量
- 辅助指标：所有可用指标
- 要求：2个指标一致即可生成信号

### 平衡型策略（默认）
- 综合所有指标评分
- 加权计算总体信号
- 根据强度决定操作建议

## 注意事项

### 指标局限性
1. **滞后性**: 技术指标基于历史数据，存在滞后
2. **失效性**: 在极端行情中指标可能失效
3. **冲突性**: 不同指标可能给出矛盾信号

### 使用建议
1. **结合基本面**: 技术分析应与基本面分析结合
2. **风险控制**: 设置止损止盈，控制仓位
3. **持续优化**: 根据市场变化调整参数
4. **多时间框架**: 同时观察多个时间框架的信号

### 验证方法
1. **回测**: 使用历史数据验证策略有效性
2. **模拟交易**: 先进行模拟交易验证
3. **小仓位测试**: 实盘时先用小仓位测试

## 扩展指标

### 可添加的指标
1. **KDJ指标**: 随机指标，反映超买超卖
2. **OBV指标**: 能量潮，量价关系指标
3. **CCI指标**: 顺势指标，判断趋势强度
4. **ATR指标**: 平均真实波幅，衡量波动性

### 添加方法
在 `signals.py` 中添加新的指标计算方法，并在 `generate_signals` 函数中集成。
